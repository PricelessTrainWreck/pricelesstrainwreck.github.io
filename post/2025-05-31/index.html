Displacement maps make mesh triangles extruded from a texture. That will have depth that is indistinguishable from a separate object – you can even do collision against displacement maps if you really want.

The thing you can’t do with displacement maps is have undercuts. Like:


     /------\            /------\

    |        |          |        |

    |        |           \      /

____|        |____________\    /______

   displacement       not displacement
